# Example: ChatGPT Development Request

## User Request

把這個故事整理成 Codex Canon，之後要讓 Codex 續寫對白本。

## Skill Response Pattern

1. Identify medium and current stage.
2. Extract existing Canon.
3. Separate confirmed facts from assumptions.
4. Build character, relationship, timeline, and style rules.
5. Produce a Markdown Canon file.
6. Add Codex-specific execution instructions.

## Example Output Sections

- Project Overview
- Core Premise
- Character Rules
- Timeline Rules
- Forbidden Deviations
- Codex Instructions
