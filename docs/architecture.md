# Architecture

## Goal
Convert a personal AI-assisted writing workflow into a reusable repository structure.

## Layers
### 1. Development
Used to define premise, characters, worldbuilding, themes, beats, and outline.

### 2. Generation
Used to produce scene drafts, continuations, and alternate versions.

### 3. Critique
Used to analyze structure, pacing, logic, emotional clarity, and consistency.

### 4. Human Decision Layer
Used to accept, reject, merge, or rewrite outputs.

## Principle
The repository should optimize controllable generation, not unconstrained improvisation.
