# Narrative Spec

Narrative Spec is the ruleset that constrains model output.

## Purpose
- preserve character consistency
- preserve world logic
- reduce hallucination
- keep tone stable across long-form writing

## Recommended Sections
1. Character invariants
2. World constraints
3. Plot prohibitions
4. Tone and style rules
5. Format rules
6. Continuity rules
7. Allowed flexibility

## Example Rules
- Do not introduce major new characters without explicit approval.
- Do not change the protagonist's core motivation.
- Do not resolve the main conflict before the designated turning point.
- Keep dialogue concise and playable.
