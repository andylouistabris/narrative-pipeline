# Andy AI Screenwriting Pipeline Skill

這是一套將「ChatGPT + Codex + GitHub」寫作流程工程化的 Skill 範本。

## 使用方式

1. 將整個 `andy-ai-screenwriting-pipeline` 資料夾放進可讀取 Skills 的位置。
2. 確認資料夾內有 `SKILL.md`。
3. 在 ChatGPT 或 Codex 中要求：

   「依照 Andy AI Screenwriting Pipeline，把這段討論整理成 Codex Canon。」

   或：

   「依照 Andy AI Screenwriting Pipeline，幫我建立 Codex 任務檔。」

## 建議用途

- 劇本開發
- 小說改編
- 舞台劇改編
- 音樂劇改編
- Canon 建立
- Codex 自動寫作任務
- 多模型評析回饋

## 核心觀念

Canon 是給 AI 的法律，不是給讀者看的設定集。

Skill 裡面應該放可執行規則，不是創作心得。
